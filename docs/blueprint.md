# **App Name**: GoldenBell Squad

## Core Features:

- Random Selection Games: Engage in playful 'Ladder Draw', 'Receipt Roulette', or 'Tap Survival' games to determine the Lunchtime Hero.
- Participant Management: Add and remove participants, choosing from humorous character types like 'Penniless Intern' or 'Corporate Card Hunter'.
- Dynamic Game UI & Sounds: Experience each game mode with lively 8-bit sound effects, tension-building animations, and celebratory visuals for results.
- Hero Commendation Generator: Utilize an AI tool to generate a unique, humorous 'Hero's Commendation' message for the winner, enhancing the celebratory mood.
- Results Display & Sharing: Display the 'GoldenBell Hero' with festive animations and a custom message, with an option to share results.
- In-app Ad Integration: Feature 'lunchtime special' banners and offer a 'reroll' option after watching a reward ad.

## Style Guidelines:

- Color Palette: A dark background sets a sophisticated yet playful stage, punctuated by vibrant, almost neon, accents inspired by retro 8-bit aesthetics and urban graffiti art. The primary color, a vivid magenta-purple (#F53DDC), adds a jolt of energetic fun, while an analogous bright purple (#C866FF) serves as an accent, ensuring high contrast and a contemporary feel. The subtle purplish-grey background (#2E1E2C) provides a calming anchor for the vibrant elements.
- Font: 'Poppins' (sans-serif) for both headlines and body text, lending a geometric, precise, and contemporary aesthetic that aligns with the hip, avant-garde style. Its clean lines ensure readability for all in-app text and integrate well with the graphic, bold visual elements.
- Icons: A blend of 8-bit pixel art for game elements and clean, bold vector icons for user interface actions. Icons like bento boxes, coffee cups, and corporate cards will float playfully in the background, reinforcing the app's theme with a humorous, almost cartoonish quality.
- Layout: Intuitive and direct, with prominent call-to-action buttons for game modes. The main screen provides a lively entry point with animated background elements. Participant selection utilizes character cards, and result screens are visually impactful, focusing on the winning hero with dynamic overlays. A dedicated banner space at the bottom ensures discrete advertising integration.
- Animations: Lively, short bursts of motion complement the 8-bit sound design. Introductory sequences feature popping text and floating objects. Game mechanics like the ladder draw will animate quickly with rhythmic pulses, and the roulette will have a decelerating spin with click sounds to build tension. The winner's announcement is marked by showering golden particles, character crown animations, and dramatic sound cues, celebrating their 'heroic' contribution.
